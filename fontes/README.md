# 📚 Fontes Utilizadas no Projeto

Este projeto foi desenvolvido com base nos seguintes materiais:

## 📘 Apostila – Educação Financeira
Material com foco em:
- Endividamento
- Planejamento financeiro
- Controle orçamentário
- Administração de crises financeiras

## 📘 Básico em Finanças Pessoais
Conteúdo introdutório sobre:
- Orçamento pessoal
- Matemática financeira
- Juros simples e compostos
- Uso consciente do crédito

## 📘 Caderno de Educação Financeira – Banco Central
Material oficial do Banco Central do Brasil abordando:
- Cidadania financeira
- Consumo consciente
- Poupança e investimento
- Prevenção e proteção financeira
