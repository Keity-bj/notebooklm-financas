# 📚 Projeto NotebookLM – Educação Financeira

Projeto desenvolvido como atividade prática da DIO.

## 🎯 Objetivo

Construir um caderno temático utilizando o NotebookLM com foco em fundamentos de educação financeira, organização orçamentária e planejamento pessoal.

## 📂 Estrutura do Projeto

- fontes/ → materiais utilizados como base teórica
- objetivos.md → metas de estudo
- perguntas.md → perguntas estratégicas
- prompts.md → comandos utilizados no NotebookLM
- miniguia.md → resumo final com glossário e respostas

## 📖 Temas Abordados

- Planejamento financeiro
- Endividamento
- Juros simples e compostos
- Orçamento familiar
- Poupança e investimento
- Consumo consciente
